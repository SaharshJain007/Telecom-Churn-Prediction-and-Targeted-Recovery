# Telecom-Churn-Prediction-and-Targeted-Recovery
This repository contains a machine-learning-based predictive model aimed at identifying telecom customer churn. It includes a deployed Streamlit simulation designed to translate predictive data into actionable business strategies by optimizing for maximum profit. This can be run either through a software like VS Code or through a CloudFlare Tunnel. It also includes the dataset used in this project and a PPT to learn about the project quickly.
